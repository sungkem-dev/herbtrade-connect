## Tentang Herblocx

Herblocx adalah platform marketplace untuk produk herbal yang memanfaatkan teknologi blockchain untuk transparansi dan keamanan transaksi. Platform ini menghubungkan pembeli dengan penjual produk herbal dengan fitur:

- **Autentikasi**: Register, Login, Logout
- **Shopping**: Browse produk, keranjang belanja, pembelian langsung
- **Buyer Dashboard**: Riwayat pesanan, tracking, requests
- **Seller Dashboard**: Kelola produk, pesanan, analytics, withdraw
- **Blockchain Integration**: Wallet connect, transparansi transaksi
